 Duelist Pack: Aster Phoenix  is a  Duelist Pack  in the  Yu-Gi-Oh! Official Card Game  ( OCG ) and  Yu-Gi-Oh! Trading Card Game  ( TCG ). It is the fifth set in the Duelist Pack series, following  Duelist Pack: Zane Truesdale .
 In the Korean  OCG ,  Duelist Pack: Aster Phoenix  was first released on February 2, 2008 as  Duelist Pack: Aster Phoenix Special Edition , before being released on April 15, 2008.

 Features 
 The set features cards used by  Aster Phoenix  from the Yu-Gi-Oh! GX series. The set contains 25 reprints and 5 new cards.
 
 Breakdown 
 In the  OCG , each pack of  Duelist Pack: Aster Phoenix  contains 5 cards and each box contains 15 packs. In the  TCG , each pack of  Duelist Pack: Aster Phoenix  contains 6 cards and each box contains 30 packs. There are 30 cards in total. The number of card of each rarity are as follows:
