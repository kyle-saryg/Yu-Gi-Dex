 Legendary Hero Decks 
 
 
 
 
 
 
 Legendary Hero Decks  is a  TCG -exclusive set that contains 3 pre-constructed Decks, and is the fourth set in the  Legendary Decks  series.

 Breakdown 
 Each  Legendary Hero Decks  box set contains 150 cards: 
 
 Three 40-card Decks with 10-card Extra Decks.
 1 Deck based on  Aster Phoenix 's " Destiny HERO " monsters 
 1 Deck based on  Team Ragnarok 's " Aesir " monsters. 
 1 Deck based on  Yuto 's " Phantom Knights " cards. 
 Each Deck contains 5  Ultra Rares  and 45  Commons .
