 Duelist Pack: Yuma  is a  Duelist Pack  in the  Yu-Gi-Oh! Official Card Game  ( OCG ). It is the twelfth set in the  OCG ' s Duelist Pack series, following  Duelist Pack: Crow . It is the first Duelist Pack for a character from the  ZEXAL  anime series.

 Features 
 The set features a lot of the latest cards from the  Yu-Gi-Oh! ZEXAL  anime used by  Yuma Tsukumo . It also contains  support  and members for the " Gagaga " and " Number "  archetypes . It contains 5 new cards along with 25 reprints.
 
 Breakdown 
 Each pack of  Duelist Pack: Yuma  contains 5 cards and each box contains 15 packs. There are 30 cards in total. The number of card of each rarity are as follows:
