 Pharaoh's Servant  is a Japanese  Booster Pack  in the  Yu-Gi-Oh! Official Card Game  ( OCG ). It is the second set in the  OCG Series 2 .
 In other languages, this set and  Magic Ruler  are amalgamated to form the set  Spell Ruler , which was previously also known as  Magic Ruler . This set shares its name with a different set called  Pharaoh's Servant , which is available in other languages.

 Features 
 Pharaoh's Servant  contains a number of cards from the  Duelist Kingdom  arc, including the  Toon Monsters , which debut in the  OCG  in this set. It contains a number of cards which were previously exclusive to the video games, including the  Ritual Monsters  " Crab Turtle ", " Performance of Sword " and " Hungry Burger ". 
 It supports all six  Attributes  with the  Elemental Searchers  and  Attribute Fields .
 Other highly sought after cards include " Cyber Jar " and " Megamorph ".
 
 Breakdown 
 Each pack contains 5 cards and each box contains 30 packs.
 The sect contains 52 cards, or 56 in the  Master Set . This is comprised of:
