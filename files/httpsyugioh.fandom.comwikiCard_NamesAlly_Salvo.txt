 The unofficial Arabic name given (قنبلة الأنصار  Qunbulatu l-ʾAnṣār ) translates to  Allies' Bomb . 
 The unofficial Greek name given ( Συμμαχική Βόμβα ) is a translation of the Japanese name of this card.
 The English name of this card can be translated in Greek as " Συμμαχικό Βλήμα Κανονιού " (approximation).
