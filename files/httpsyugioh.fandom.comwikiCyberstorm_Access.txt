 Cyberstorm Access 
 
 
 
 
 
 
 Cyberstorm Access  is a  Booster Pack  in the  Yu-Gi-Oh! Official Card Game  ( OCG ). It is the twelfth and final set in the  OCG ' s  11th series , following  Photon Hypernova .

 Features 
 In the first print runs of Cyberstorm Access boxes, includes an  Cyberstorm Access +1 Bonus Pack 
 Includes new members/support for the " Cyber ", " Code Talker " and " Cynet "  archetypes . 
 Introduces the first support for the " Firewall " archetype. 
 Breakdown
