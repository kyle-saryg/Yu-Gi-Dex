 Yu-Gi-Oh! Rush Duel Starter Deck Set - Yuga vs. Luke 
 
 
 
 
 
 
 Starter Deck Set - Yuga vs. Luke  is a set in the  Yu-Gi-Oh! Rush Duel  card game. It packages  Yu-Gi-Oh! Rush Duel Starter Deck: Yuga - Open! Sevens Road!!  and   Yu-Gi-Oh! Rush Duel Starter Deck: Luke - Explosive Conquest! Dragears!! , and alongside them introduce the  Yu-Gi-Oh! Rush Duel  card game.
