 Card descriptions 
 
 
 
   English 
 
 
If a  Chain  (of 2 or more  Links ) occurred in your  Main Phase 1 , this card can  attack  twice during the  Battle Phase  this  turn . 
 
 
 
 
 
   French 
 
 
 Si une Chaîne (de 2 Blocks ou plus) se produit pendant votre Main Phase 1, cette carte peut attaquer deux fois durant la Battle Phase ce tour. 
 
 
 
 
 
   German 
 
 
 Falls eine Kette (aus 2 oder mehr Gliedern) in deiner Main Phase 1 auftrat, kann diese Karte in diesem Spielzug während der Battle Phase zweimal angreifen. 
 
 
 
 
 
   Italian 
 
 
 Se si è verificata una Catena (con 2 o più Anelli) durante la tua Main Phase 1, questa carta può attaccare due volte durante la Battle Phase questo turno. 
 
 
 
 
 
   Spanish 
 
 
 Si una Cadena (de 2 o más Eslabones) ocurrió durante tu Main Phase 1, esta carta puede atacar dos veces en la Battle Phase de este turno. 
 
 
 
 
 
   Japanese 
 
 
 このカードがフィールド上に表側表示で存在する時に自分ターンのメインフェイズ１でチェーンが発生した場合、このターンのみこのカードは１度のバトルフェイズ中に２回攻撃する事ができる。
