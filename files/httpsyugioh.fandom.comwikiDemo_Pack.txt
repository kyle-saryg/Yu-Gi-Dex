 Demo Deck 
 Demo Deck 2015 
 
 
 
 
 
 Demo Pack  is a pack released from February 12, 2011 to March 13, 2011. Attendees unfamiliar with the game had the opportunity to participate in a hands-on demonstration of how to play the game and ask questions. Using this specially designed demo pack and a guide, the Yu-Gi-Oh! specialists would expertly teach the basics of the game to aspiring Duelists.
 
 Breakdown 
 Each pack contains 20 cards, all  Common .
