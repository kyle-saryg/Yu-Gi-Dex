 Card descriptions 
 
 
 
   English 
 
 
2  Level  4  monsters When this card is  Xyz Summoned : You can  discard  any number of cards; increase this card's  Rank  by the number of discarded cards.  Once per turn , while this card has  Xyz Material : You can target 1  monster  in your  GY ;  detach  all materials from this card, and if you do, attach that target to this card as material. When a monster is attached by this effect, this card's  Type  and  Attribute  become the same as that monster's original Type and Attribute. 
 
 
 
 
 
   French 
 
 
 2 monstres de Niveau 4 Lorsque cette carte est Invoquée par Xyz : vous pouvez défausser un nombre de votre choix de cartes ; augmentez le Rang de cette carte du nombre de cartes défaussées. Une fois par tour, tant que cette carte a du Matériel Xyz : vous pouvez cibler 1 monstre dans votre Cimetière ; détachez tous les Matériels de cette carte, et si vous le faites, attachez la cible à cette carte comme Matériel. Lorsqu'un monstre est attaché par cet effet, le Type et Attribut de cette carte deviennent les mêmes que le Type et Attribut d'origine du monstre.
