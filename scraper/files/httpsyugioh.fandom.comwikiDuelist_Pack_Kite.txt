 Duelist Pack: Kite  is a  Duelist Pack  in the  Yu-Gi-Oh! Official Card Game  ( OCG ). It is the thirteenth set in the  OCG ' s Duelist Pack series, following  Duelist Pack: Yuma .

 Features 
 The set features a lot of the latest cards from the  Yu-Gi-Oh! ZEXAL  anime used by  Kite Tenjo .
 It introduces the first support for the " Galaxy " archetype, as well as additional  support  for the " Photon "  archetypes ,  LIGHT   monsters  and  Xyz Summoning . It contains 12 new cards along with 18 reprints.
 
 Breakdown 
 Each pack of  Duelist Pack: Kite  contains 5 cards and each box contains 15 packs. There are 30 cards in total. The number of card of each rarity are as follows:
