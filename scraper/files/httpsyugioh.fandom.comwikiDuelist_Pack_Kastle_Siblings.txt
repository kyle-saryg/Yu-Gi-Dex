 Duelist Pack: Kastle Siblings  is a  Duelist Pack  in the  Yu-Gi-Oh! Official Card Game  ( OCG ). It is the fifteenth set in the  OCG ' s Duelist Pack series, following  Duelist Pack: Yuma 2: Gogogo & Dododo .

 Features 
 The set features a lot of the latest cards from the  Yu-Gi-Oh! ZEXAL  anime used by  Ryoga Kamishiro  and  Rio Kamishiro .
 
 Breakdown 
 Each pack of  Duelist Pack: Kamishiro Siblings  contains 5 cards and each box contains 15 packs. There are 30 cards in total. The number of card of each rarity are as follows:
