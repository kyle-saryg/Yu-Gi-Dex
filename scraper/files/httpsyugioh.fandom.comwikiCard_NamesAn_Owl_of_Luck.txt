 Names 
 
 The unofficial  Greek  name given ( Μια Τυχερή Κουκουβάγια ) translates to " A Lucky Owl ".
 The Japanese name of this card ( Owl Informs of Luck ) can be translated in Greek as " Κουκουβάγια Πληροφορεί για την Τύχη ". 
 The Spanish name of this card has an error, as it should be "' Un B ú ho de Suerte " instead of " Un B u ho de Suerte ".
