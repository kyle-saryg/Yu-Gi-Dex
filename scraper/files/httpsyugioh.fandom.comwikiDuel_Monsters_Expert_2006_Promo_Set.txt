 Duel Monsters Expert 2006 Promo Set 
 
 
 
 
 
 
 This set is the Deluxe Edition of the Game Boy Advance game  Yu-Gi-Oh! Duel Monsters Expert 2006 , released only in Japan on February 23, 2006. In addition to the game, this set contained 1x  Structure Deck: Warrior's Triumph , 1x  Structure Deck: Spellcaster's Judgment , 1x " Yu-Gi-Oh! Online  Installation CD" + 1x "Duelpass Phase 3" and 4 Ultimate Rare cards.
 
 Breakdown 
 The promo pack consists of 4 Effect Monsters (the main monsters from the first 4 Structure decks) in Ultimate Rare.
