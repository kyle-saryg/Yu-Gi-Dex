 Duelist Pack: Yusei  is a  Duelist Pack  in the  Yu-Gi-Oh! Official Card Game  ( OCG ) and  Yu-Gi-Oh! Trading Card Game  ( TCG ). It is the eighth set in the Duelist Pack series, following  Duelist Pack: Jesse Anderson . It is the first Duelist Pack for a character from the  5D's  anime series.

 Features 
 The set features cards used by the  Yu-Gi-Oh! 5D's  main protagonist  Yusei Fudo  in the anime.
 
 Breakdown 
 In the  OCG , each pack of  Duelist Pack: Yusei  contains 5 cards and each box contains 15 packs. In the  TCG , each pack of  Duelist Pack: Yusei  contains 5 cards and each box contains 30 packs. There are 30 cards in total. The number of card of each rarity are as follows:
