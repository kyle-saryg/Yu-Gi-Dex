 Premium Pack Vol.11  is a Korean pack, containing a number of cards that were released as promos in the Japanese  OCG , that were previously unreleased in Korean.
 
 Breakdown 
 Each pack contains 5 cards. Each box contains 20 packs. The set contains 41 cards:
 
 5  Secret Rares 
 8  Ultra Rares 
 28  Super Rares
